# postgresql-sandbox

## Install

```
docker-compose up
```

## PostgreSQL

```
docker-compose exec postgres bash
```

```
psql -U postgres
```

## tbls

```
cd /home
wget https://github.com/k1LoW/tbls/releases/download/v1.15.4/tbls_v1.15.4_linux_amd64.tar.gz
tar -zxvf tbls_v1.15.4_linux_amd64.tar.gz
./tbls doc
```
